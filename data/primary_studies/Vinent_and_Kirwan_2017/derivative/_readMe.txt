Dataset: Upper Phillips Creek soil organic content and bulk density April, 2017

Authors: Vinent, O.D. and ML. Kirwan

Any use of this dataset must include a citation. doi:10.6073/pasta/0f1cceb5f013643be08dbc5386f073ac

The data release contains tidal wetland soil carbon profiles. 

Vinent_and_Kirwan_2017_cores.csv - Contains positional and descriptive information on core locations.
Vinent_and_Kirwan_2017_depthseries.csv - Contains raw depth-series information for all cores.
Vinent_and_Kirwan_2017_methods.csv - Contains information on materials and methods broken down by study

Vinent_and_Kirwan_2017_study_citations.bib and Vinent_and_Kirwan_2017_study_citations.csv - Contain bibliographic information for publication associated with this data release.
