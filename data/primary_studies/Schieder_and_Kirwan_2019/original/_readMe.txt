Title: Sea-level driven acceleration in coastal forest retreat

Authors: Nathalie W. Schieder; Matthew L. Kirwan

The data release contains tidal wetland soil carbon profiles. The data itself is housed in three separate .csv files which can by joined by the core_id and/or site_id attributes. 

To cite this data, please use the original DOI: 10.25573/serc.25259983

schieder_and_kirwan_2019_methods.csv - Contains information on materials and methods broken down by study 
schieder_and_kirwan_2019_cores.csv - Contains positional and descriptive information on core locations
schieder_and_kirwan_2019_depthseries.csv - Contains raw depth-series information for all cores
schieder_and_kirwan_2019_study_citations.csv - Contains bibliographic information for this data release
