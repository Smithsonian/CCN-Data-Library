Dataset: Sediment Carbon Content from three Maine Salt Marshes 1993

Authors: Robert Vincent <rvincent@mit.edu>

Any use of this dataset must include a citation. The DOI: 10.25573/serc.23960793

The data release contains tidal wetland soil carbon profiles. The data itself is housed in four separate .csv files which can be joined by the core_id and/or site_id attributes. 

Vincent_2023_materials_and_methods.csv - Contains information on materials and methods broken down by study 
Vincent_2023_cores.csv - Contains positional and descriptive information on core locations.
Vincent_2023_depthseries.csv - Contains raw depth-series information for all cores.
Vincent_2023_sites.csv - Contains information on the sites of the coring locations.
Vincent_2023_species.csv - Contains taxonomic or descriptive information of species present at coring locations.
Vincent_2023_impacts.csv - Contains descriptive information on anthropogenic impacts associated with coring sites. 

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
