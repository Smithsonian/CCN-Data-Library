Dataset: Recent Acceleration of Coastal Wetland Accretion Along the U.S. East Coast

Authors: Nathaniel B. Weston <nathaniel.weston@villanova.edu>, Elise Rodriguez, Brian Donnelly, Elena Solohin, Kristen Jezycki, Sandra Demberger, Lori Sutter, James T. Morris, Scott Neubauer, and Christopher B. Craft.

Any use of this dataset must include a citation. The DOI: 10.25573/serc.13043054

The data release contains tidal wetland soil carbon profiles. The data itself is housed in four separate .csv files which can be joined by the core_id and/or site_id attributes. 

Weston_et_al_2023_materials_and_methods.csv - Contains information on materials and methods broken down by study 
Weston_et_al_2023_cores.csv - Contains positional and descriptive information on core locations.
Weston_et_al_2023_depthseries.csv - Contains raw depth-series information for all cores.
Weston_et_al_2023_species.csv - Contains information on the dominant plant species at coring locations.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.pdf - Contains descriptive metadata, including a detailed breakdown of the methods and equations used for the study.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.

Weston_et_al_2023.bib - Contains bibliographic information for publication associated with this data release.
