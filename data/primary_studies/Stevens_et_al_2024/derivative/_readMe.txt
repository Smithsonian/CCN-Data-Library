Title: Sediment Accumulation in Salt Marshes Across the Southeastern United States

Authors: Luke Stevens, D. Reide Corbett, Stephen J. Culver

The data release contains tidal wetland soil carbon profiles. The data itself is housed in five separate .csv files which can by joined by the core_id and/or site_id attributes. 

To use this data, please cite the following DOI: 10.25573/serc.25289635

stevens_et_al_2024_methods.csv - Contains information on materials and methods broken down by study 
stevens_et_al_2024_cores.csv - Contains positional and descriptive information on core locations
stevens_et_al_2024_depthseries.csv - Contains raw depth-series information for all cores
stevens_et_al_2024_sites.csv - Contains positional information on site locations
stevens_et_al_2024_species.csv - Contains information on the dominant plant species at coring locations
stevens_et_al_2024_impacts.csv - Contains information on the anthropogenic impacts of the site locations
stevens_et_al_2024_study_citations.csv - Contains bibliographic information for publication associated with this data release
