Dataset: Sediment depth and accretion shape belowground mangrove carbon stocks across a range of climatic and geologic settings. 

Authors: Matthew T. Costa <mtcosta@ucsd.edu>, Exequiel Ezcurra, Paula Ezcurra, Pelayo Salinas-de-Leon, Benjamin Turner, James Leichter, and Octavio Aburto-Oropeza

The data release contains tidal wetland soil carbon profiles. The data itself is housed in five separate .csv files which can by joined by the core_id and/or site_id attributes. 

costa_et_al_2023_methods.csv - Contains information on materials and methods broken down by study 
costa_et_al_2023_cores.csv - Contains positional and descriptive information on core locations
costa_et_al_2023_depthseries.csv - Contains raw depth-series information for all cores
costa_et_al_2023_sites.csv - Contains positional information on site locations
costa_et_al_2023_species.csv - Contains information on the dominant plant species at coring locations
costa_et_al_2023_associated_publication.bib - Contains bibliographic information for publication associated with this data release

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
