Dataset: Vibracore and Tree Stump Data from the Marsh Near Mary Hammock, McIntosh County, GA

Authors: John Turck

Any use of this dataset must include a citation. The DOI: 10.6073/pasta/4541ae084d807962b8c331eea61908bd

The data release contains tidal wetland soil carbon profiles. The data itself is housed in four separate .csv files which can be joined by the core_id and/or site_id attributes. 

Turck_2014_methods.csv - Contains information on materials and methods broken down by study.
Turck_2014_cores.csv - Contains positional and descriptive information on core locations.
Turck_2014_depthseries.csv - Contains raw depth-series information for all cores.
Turck_2014_species.csv - Contains information on the dominant plant species at coring locations.

Turck_2014_study_citations.bib and Turck_2014_study_citations.csv - Contain bibliographic information for publication associated with this data release.
