Dataset:  A Preliminary Sediment Budget for the Corsica River (MD): Improved Estimates of Nitrogen Burial and Implications for Restoration

Authors: Cindy M. Palinkas, Jeffrey Cornwell

Any use of this dataset must include a citation. The DOI: 10.25573/serc.24467977

The data release contains sediment carbon profiles from tidal wetlands. The data itself is housed in four separate .csv files which can by joined by the core_id and/or site_id attributes. 

Palinkas_and_Cornwell_2024_methods.csv - Contains methodological information on the soil core collection and processing.
Palinkas_and_Cornwell_2024_sites.csv - Contains site-level information.
Palinkas_and_Cornwell_2024_depthseries.csv - Contains raw depth-series information for all cores.
Palinkas_and_Cornwell_2024_cores.csv - Contains core, or sample-level information for all cores.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
