Dataset: A Synthesis of South African Estuaries

Synthesis Curators: Anesu Machite, Jaqueline Raw, Janine Adams

Any use of this dataset must include a citation. The DOI: 10.25573/serc.24394426 

The data release contains organic matter profiles from tidal wetlands. The data itself is housed in five separate .csv files which can by joined by the core_id and/or site_id attributes

Machite_et_al_2024_methods.csv - Contains methodological information on the soil core collection and processing.
Machite_et_al_2024_sites.csv - Contains site-level information.
Machite_et_al_2024_depthseries.csv - Contains raw depth-series information for all cores.
Machite_et_al_2024_cores.csv - Contains core, or sample-level information for all cores.
Machite_et_al_2024_species.csv - Contains taxonomic information for all sites.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.

dataset_authors.csv - Contains information and affiliation for all associated dataset authors for each study and thesis represented.
