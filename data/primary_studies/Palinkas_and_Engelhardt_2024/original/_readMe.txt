Dataset:Spatial and temporal patterns of modern sedimentation in a tidal freshwater marsh

Authors: Cindy M. Palinkas, Katharina A.M. Engelhardt

Any use of this dataset must include a citation. The DOI: 10.25573/serc.24470152

The data release contains organic matter profiles from tidal wetlands. The data itself is housed in six separate .csv files which can by joined by the core_id and/or site_id attributes. 

Palinkas_and_Engelhardt_2024_methods.csv - Contains methodological information on the soil core collection and processing.
Palinkas_and_Engelhardt_2024_sites.csv - Contains site-level information.
Palinkas_and_Engelhardt_2024_depthseries.csv - Contains raw depth-series information for all cores.
Palinkas_and_Engelhardt_2024_cores.csv - Contains core, or sample-level information for all cores.
Palinkas_and_Engelhardt_2024_species.csv - Contains taxonomic information for all sites.
Palinkas_and_Engelhardt_2024_impacts.csv -  Contains site-level information on anthropogenic impacts. 

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
