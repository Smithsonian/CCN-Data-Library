The data release contains tidal wetland soil carbon profiles. The data itself is housed in separate .csv files which are relational via StudyID and/or SampID. 

Janousek_et_al_2025_estuaries.csv - Contains information on the estuary setting for the coring locations.
Janousek_et_al_2025_cores.csv - Contains positional and descriptive information on core locations.
Janousek_et_al_2025_depthseries.csv - Contains raw depth-series information for all core profiles.
Janousek_et_al_2025_sources.csv - Contains bibliographic information for each original source as well as the materials and methods used by each study.


Janousek_et_al_2025_metadata.csv - Contains a full suite of descriptive metadata with all attributes, variables, and units defined.
