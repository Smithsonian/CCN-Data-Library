Dataset: Head of tide bottleneck of particulate material transport from watersheds to estuaries

Authors: Scott H. Ensign, Gregory B. Noe, Cliff R. Hupp, and Katherine J. Skalak.

Any use of this dataset must include a citation. The DOI: 10.25573/serc.13483332

The data release contains tidal wetland soil carbon profiles. The data itself is housed in three separate .csv files which can be joined by the core_id and/or site_id attributes. 

ensign_et_al_2020_materials_and_methods.csv - Contains information on materials and methods broken down by study 
ensign_et_al_2020_cores.csv - Contains positional and descriptive information on core locations.
ensign_et_al_2020_depthseries.csv - Contains raw depth-series information for all cores.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.

map.html - Is a map widget showing the geographic coverages described in the metadata. It can be accessed on its own or through metadata.html.

custom.css - Contains display information for metadata.html.
