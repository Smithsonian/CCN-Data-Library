Dataset: Evaluation of carbon in the National Forest of Térraba-Sierpe, Costa Rica

Authors: Miguel Cifuentes-Jara (miguel.cifuentes.jara@outlook.com) and Danilo Torres Gómes

Any use of this dataset must include a citation. The DOI: 10.25573/serc.26999278

The data release contains biomass and soil carbon profiles from tidal wetlands. The data itself is housed in separate .csv files which can by joined by relational attributes (ex. site_id and plot_id).

Cifuentes_and_Torres_2024_methods.csv - Contains methodological information on the soil core collection and processing.
Cifuentes_and_Torres_2024_plots.csv - Contains plot-level information.
Cifuentes_and_Torres_2024_depthseries.csv - Contains raw depth-series information for all cores.
Cifuentes_and_Torres_2024_plants.csv - Contains information on the above and belowground plant mass and community composition of each plot.
Cifuentes_and_Torres_2024_debris.csv - Contains information on the woody debris measured along transects in each plot.
Cifuentes_and_Torres_2024_understory.csv - Contains information on the understory coverage and associated plant mass within each plot.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
