Title: Sea Level-Driven Marsh Migration Results in Rapid Net Loss of Carbon

Authors: Alexander Smith, Matthew Kirwan

The data release contains tidal wetland soil core information that can be joined across site ID and core ID.

smith_and_kirwan_2021_study_citations.csv - Contains bibliographic information for the article associated with this release
smith_and_kirwan_2021_methods.csv - Contains information on materials and methods broken down by study 
smith_and_kirwan_2021_cores.csv - Contains positional and descriptive information on core locations
smith_and_kirwan_2021_depthseries.csv - Contains raw depth-series information for all cores
