Dataset: Carbon Storage Increases with Site Age as Created Salt Marshes Transition to Mangrove Forests in Tampa Bay, Florida (USA)


Authors: Emma E. Dontis, Kara R. Radabaugh, Amanda R. Chappel, Christine E. Russo & Ryan P. Moyer 

The data release contains tidal wetland biomass information.

Any use of this dataset must include a citation. The DOI: https://doi.org/10.25573/serc.24467947.v1


dontis_et_al_2020_plant.csv - Contains information on plant species, height, and weight
dontis_et_al_2020_plot_summary.csv - Contains positional and descriptive information of plant plot locations  
dontis_et_al_2020_study_citations.bib - Contains bibliographic information for this data release
dontis_et_al_2020_study_citations.csv - Contains bibliographic information for this data release
dontis_et_al_2020_allometric_eq.csv - Contains information on species specific allometric equations 
dontis_et_al_2020_impacts.csv - Contains anthropological impact information on site locations
dontis_et_al_2020_species.csv - Contains information on the dominant plant species at coring locations
