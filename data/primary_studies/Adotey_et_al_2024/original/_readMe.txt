Dataset: Carbon Stock Assessment in the Kakum and Amanzule Estuary Mangrove Forests, Ghana

Authors: Joshua Adotey <joshua.adotey@ucc.edu.gh>, Denis Worlanyo Aheto, John Blay, Emmanuel Acheampong

Any use of this dataset must include a citation. The DOI: 10.25573/serc.25148561

The data release contains biomass and soil carbon profiles from tidal wetlands. The data itself is housed in separate .csv files which can by joined by relational attributes (ex. site_id and plot_id).

Adotey_et_al_2024_plots.csv - Contains positional and descriptive information of plant plot locations. 
Adotey_et_al_2024_plants.csv - Contains information on the aboveground tree allometry and species composition of each surveyed plot.
Adotey_et_al_2024_depthseries.csv - Contains raw depth-series information for all soil cores
Adotey_et_al_2024_methods.csv - Contains methodological information on the soil core collection and processing. 

Adotey_et_al_2024_study_citations.bib - Contains bibliographic information for publication associated with this data release.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
