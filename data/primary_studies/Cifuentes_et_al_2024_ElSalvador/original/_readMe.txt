Dataset: Land cover dynamics and carbon stocks in El Salvador mangroves

Authors: Miguel Cifuentes-Jara, Danilo Torres Gómez, Oscar Molina Lara, Sergio Velásquez Mazariegos, Carlos Giovanni Rivera, Javier Magaña Rivas

Any use of this dataset must include a citation. The DOI: 10.25573/serc.25828603

The data release contains biomass and soil carbon profiles from mangrove ecosystems. The data itself is housed in separate .csv files which can by joined by relational attributes (ex. site_id and plot_id).

Cifuentes_et_al_2024_ElSalvador_methods.csv - Contains methodological information on the soil core collection and processing.
Cifuentes_et_al_2024_ElSalvador_plots.csv - Contains plot-level information.
Cifuentes_et_al_2024_ElSalvador_depthseries.csv - Contains raw depth-series information for all cores.
Cifuentes_et_al_2024_ElSalvador_plants.csv - Contains information on the above and belowground plant mass and community composition of each plot.
Cifuentes_et_al_2024_ElSalvador_debris.csv - Contains information on the woody debris measured along transects in each plot.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
