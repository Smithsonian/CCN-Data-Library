Dataset: Dataset:Carbon Sequestration and Nutrient (Nitrogen, Phosphorus) Accumulation in River-Dominated Tidal Marshes, Georgia, USA

Authors: Mark J. Loomis, Christopher B. Craft

Any use of this dataset must include a citation. The DOI: 10.25573/serc.24518755

The data release contains sediment carbon profiles from tidal wetlands. The data itself is housed in three separate .csv files which can by joined by the core_id and/or site_id attributes. 

Loomis_and_Craft_2023_methods.csv - Contains methodological information on the soil core collection and processing.
Loomis_and_Craft_2023_depthseries.csv - Contains raw depth-series information for all cores.
Loomis_and_Craft_2023_cores.csv - Contains core, or sample-level information for all cores.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
