Dataset: Carbon storage and accretion in Sweet Hall Marsh, Virginia

Authors: Scott Lerberg (lerbergs@vims.edu), Alex Demeo, Hank Brooks, Erin Reilly, Jennifer Connell, Katherine Kivimaki, Allyson Boggess, James Holmquist

Any use of this dataset must include a citation. The DOI: 10.25573/serc.27637311

The data release contains sediment carbon profiles from tidal wetlands. The data itself is housed in three separate .csv files which can by joined by the core_id and/or site_id attributes. 

Lerberg_et_al_2025_methods.csv - Contains methodological information on the soil core collection and processing.
Lerberg_et_al_2025_cores.csv - Contains core, or sample-level information for all cores.
Lerberg_et_al_2025_depthseries.csv - Contains raw depth-series information for all cores.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
