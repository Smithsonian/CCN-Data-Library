Dataset: Dataset: The Effect of Marsh Age on Ecosystem Function in a Rapidly Transgressing Marsh

Authors: Amy K. Langston, Daniel J. Coleman, Nathalie W. Jung, Justin L. Shawler, Alexander J. Smith, Bethany L. Williams, Serina S. Wittyngham, Randolph M. Chambers, James E. Perry, Matthew L. Kirwan

Use of the data should include the DOI: 10.25573/serc.24913215

The data release contains tidal wetland soil core information that can be joined across site ID and core ID.

langston_et_al_2022_study_citations.bib - Contains bibliographic information for this data release
langston_et_al_2022_study_citations.csv - Contains bibliographic information for this data release
langston_et_al_2022_species.csv - Contains information on the dominant plant species at coring locations
langston_et_al_2022_methods.csv - Contains information on materials and methods broken down by study 
langston_et_al_2022_cores.csv - Contains positional and descriptive information on core locations
langston_et_al_2022_depthseries.csv - Contains raw depth-series information for all cores
