Dataset: Examining Coastal Marsh Sedimentation in Northeastern North Carolina

Authors: Jessica Strand, D. Reide Corbett <corbettd@ecu.edu>

Any use of this dataset must include a citation. The DOI: 10.25573/serc.24991359

The data release contains tidal wetland soil carbon profiles. The data itself is housed in three separate .csv files which can be joined by the core_id and/or site_id attributes. 

Strand_et_al_2024_materials_and_methods.csv - Contains information on materials and methods broken down by study 
Strand_et_al_2024_cores.csv - Contains positional and descriptive information on core locations.
Strand_et_al_2024_depthseries.csv - Contains raw depth-series information for all cores.
Strand_et_al_2024_impacts.csv - Contains anthropogenic impact information on core locations.
 
 
metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
