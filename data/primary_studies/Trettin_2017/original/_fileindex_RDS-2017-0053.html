Mangrove carbon stocks in Zambezi River Delta, Mozambique

File Folder Description

Zambezi_WoodyDebris_CWD.csv \Data Comma-delimited ASCII text file containing coarse woody debris (CWD) data collected within the measurement subplots.
Zambezi_WoodyDebris_LitterandLive.csv \Data Comma-delimited ASCII text file containing litter and live ground vegetation data collected within the measurement subplots.
