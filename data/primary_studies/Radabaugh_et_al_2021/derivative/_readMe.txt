Dataset: Early indicators of stress in mangrove forests with altered hydrology in Tampa Bay, Florida, USA

Authors: Kara R. Radabaugh, Emma E. Dontis, Amanda R. Chappel, Christine E. Russo, Ryan P. Moyer

The data release contains tidal wetland soil carbon profiles. The data itself is housed in four separate .csv files which can by joined by the core_id and/or site_id attributes. 

Any use of this dataset must include a citation. The DOI: https://doi.org/10.25573/serc.23960811.v1

radabaugh_et_al_2021_methods.csv - Contains information on materials and methods broken down by study 
radabaugh_et_al_2021_cores.csv - Contains positional and descriptive information on core locations
radabaugh_et_al_2021_depthseries.csv - Contains raw depth-series information for all cores
radabaugh_et_al_2021_impacts.csv - Contains anthropological impact information on site locations
radabaugh_et_al_2021_study_citations.bib - Contains bibliographic information for this data release
radabaugh_et_al_2021_study_citations.csv - Contains bibliographic information for this data release
