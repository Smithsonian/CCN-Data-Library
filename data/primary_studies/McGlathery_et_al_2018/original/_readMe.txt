Dataset: Lead 210 profiles in sediment cores in South Bay, Virginia

Authors: McGlathery, Karen; Greiner, Jill; Gunnell, John; McKee, Brent; Oreska, Matthew; Bost, Molly

Any use of this dataset must include a citation. The DOI: 10.6073/pasta/b43fe3f3341ff2266dc50cfb8f47d026

The data release contains tidal wetland soil carbon profiles. The data itself is housed in three separate .csv files which can be joined by the core_id and/or site_id attributes. 

McGlathery_et_al_2018_methods.csv - Contains information on materials and methods broken down by study.
McGlathery_et_al_2018_cores.csv - Contains positional and descriptive information on core locations.
McGlathery_et_al_2018_depthseries.csv - Contains raw depth-series information for all cores.

McGlathery_et_al_2018_study_citations.bib and McGlathery_et_al_2018_study_citations.csv - Contain bibliographic information for publication associated with this data release.
