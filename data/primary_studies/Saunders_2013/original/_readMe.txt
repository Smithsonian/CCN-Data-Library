Dataset: Radiometric Characteristics of Soil Sediments from Shark River Slough, Everglades National Park (FCE) from 2005 and 2006

Authors: Colin Saunders

Any use of this dataset must include a citation. The DOI: 10.6073/pasta/c0cb8ff0f150e429674ecf0db15bedc5

The data release contains tidal wetland soil carbon profiles. The data itself is housed in three separate .csv files which can be joined by the core_id and/or site_id attributes. 

Saunders_2013_methods.csv - Contains information on materials and methods broken down by study.
Saunders_2013_cores.csv - Contains positional and descriptive information on core locations.
Saunders_2013_depthseries.csv - Contains raw depth-series information for all cores.

Saunders_2013_study_citations.bib and Saunders_2013_study_citations.csv - Contain bibliographic information for publication associated with this data release.
