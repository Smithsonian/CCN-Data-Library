Dataset: Carbon sequestration across Northeastern, Florida Coastal Wetlands

Authors: Claire Fell <cfell1@villanova.edu>, Therese Adgie, Samantha Chapman

Any use of this dataset must include a citation. The DOI: 10.25573/serc.15043920

The data release contains tidal wetland soil carbon profiles. The data itself is housed in four separate .csv files which can be joined by the core_id and/or site_id attributes. 

fell_et_al_2021_materials_and_methods.csv - Contains information on materials and methods broken down by study 
fell_et_al_2021_site.csv - Contains positional information on site locations.
fell_et_al_2021_cores.csv - Contains positional and descriptive information on core locations.
fell_et_al_2021_depthseries.csv - Contains raw depth-series information for all cores.
fell_et_al_2021_species.csv - Contains information on the dominant plant species at coring locations.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
