Dataset: Coastal Blue Carbon Assessment of Mangroves, Salt Marshes, and Salt Barrens in Tampa Bay, Florida, USA

Authors: Kara R. Radabaugh, Ryan P. Moyer, Amanda R. Chappel, Christina E. Powell, Ioana Bociu, Barbara C. Clark, Joseph M. Smoak

The data release contains tidal wetland soil carbon profiles. The data itself is housed in five separate .csv files which can by joined by the core_id and/or site_id attributes. 

Any use of this dataset must include a citation. The DOI: https://doi.org/10.25573/serc.23960784.v1


radabaugh_et_al_2018_methods.csv - Contains information on materials and methods broken down by study 
radabaugh_et_al_2018_cores.csv - Contains positional and descriptive information on core locations
radabaugh_et_al_2018_depthseries.csv - Contains raw depth-series information for all cores
radabaugh_et_al_2018_impacts.csv - Contains anthropological impact information on site locations
radabaugh_et_al_2018_study_citations.bib - Contains bibliographic information for this data release
radabaugh_et_al_2018_study_citations.csv - Contains bibliographic information for this data release
radabaugh_et_al_2018_plot_summary.csv - Contains positional and descriptive information of plant plot locations
