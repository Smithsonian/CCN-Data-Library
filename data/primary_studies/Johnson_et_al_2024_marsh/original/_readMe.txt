Dataset: Contributions of organic and inorganic matter to sediment volume and accretion in tidal wetlands at steady state

Authors: Beverly Johnson <bjohnso3@bates.edu>, Ashley Kulesza, Margaret Pickoff, Daniel Stames, Cailene Gunn, Brianna Karboski, and Cameron Russ

Any use of this dataset must include a citation. The DOI: 10.25573/serc.17018816

The data release contains tidal wetland soil carbon profiles. The data itself is housed in four separate .csv files which can be joined by the core_id and/or site_id attributes. 

johnson_et_al_2023_materials_and_methods.csv - Contains information on materials and methods broken down by study 
johnson_et_al_2023_cores.csv - Contains positional and descriptive information on core locations.
johnson_et_al_2023_depthseries.csv - Contains raw depth-series information for all cores.
johnson_et_al_2023_species.csv - Contains information on the dominant plant species at coring locations.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.

johnson_et_al_2023_associated_publications.csv - Contains bibliographic information for publication associated with this data release.
