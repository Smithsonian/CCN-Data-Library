Dataset: Assessment of Modern Sediment Storage in the Floodplain of the Lower Tar River, North Carolina

Authors: Dimitri Quafisi, D. Reide Corbett <corbettd@ecu.edu>

Any use of this dataset must include a citation. The DOI:10.25573/serc.25075568

The data release contains sediment profiles from tidal wetlands. The data itself is housed in four separate .csv files which can by joined by the core_id and/or site_id attributes. 

Quafisi_et_al_2024_methods.csv - Contains methodological information on the soil core collection and processing.
Quafisi_et_al_2024_sites.csv - Contains site-level information for coring sites.
Quafisi_et_al_2024_depthseries.csv - Contains raw depth-series information for all cores.
Quafisi_et_al_2024_cores.csv - Contains core, or sample-level information for all cores.

metadata.xml - Contains a full suite of descriptive metadata with all attribute and variables defined, units clarified, and geographic, temporal, and taxonomic coverages described according to Ecological Metadata Language Standards.

metadata.html - Is a simplified, visual and interactive version of metadata.xml for display purposes.
